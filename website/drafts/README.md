# Website Article Drafts

This directory is the active draft location for new article collaboration.

Use `website/drafts/draft-<slug>.md` for in-progress article plans and section drafts. Formal articles are implemented under `website/docs/<Article-Title>/<Article-Title>.md` after the draft is confirmed.

Historical drafts migrated from the old `website/docs/draft/` tree live in `website/drafts/legacy/` and should not be used as active drafts.
