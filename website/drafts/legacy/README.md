# 历史草稿目录

`website/drafts/legacy/` 是迁移前遗留的草稿目录，仍保留旧文档以便查阅和迁移，但不再作为新文章协作的 active draft 位置。

新文章草稿请使用 `website/drafts/draft-<slug>.md`。进入正式实现时，再根据文章规范写入 `website/docs/<Article-Title>/<Article-Title>.md`。

此目录位于 `website/docs/` 之外，因此不会被 Astro 文章集合加载，也不会出现在首页、文章路由与 RSS。
